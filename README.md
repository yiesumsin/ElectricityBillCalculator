# Electricity Bill Calculator

An Android application for calculating monthly electricity bills based on TNB block rates.

## Features
- Calculate electricity bills using block rate system
- Select month and year
- Apply rebate (0-5%)
- Save bills to local database
- View bill history
- Detailed bill information
- About page with student information

## Requirements
- Android 7.0 (API 24) or higher
- Internet permission (for GitHub link)

## Installation
1. Download the APK from releases
2. Enable "Install from unknown sources" if needed
3. Install and run

## How to Use
1. Select month and year
2. Enter electricity units used (kWh)
3. Select rebate percentage
4. Click "CALCULATE BILL"
5. Click "SAVE TO HISTORY" to store
6. View saved bills in "VIEW HISTORY"

## Technology Stack
- Java
- SQLite Database
- Android SDK
- Material Design
